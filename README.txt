This project involves creating a custom version of the classic game Minesweeper using stacks and queues. The focus is on implementing efficient game mechanics using these data structures while exploring problem-solving strategies for revealing zeroes and starting areas in the field.


How to compile and run the program- 1. Extract the contents of the zip file to a directory on your computer.
				    2. Using terminal or command prompt, go the directory with the contents by using the cd command.
                                    3. Use the javac command to compile the java source file as: javac Main.java
  				    4. Next, you can run the program by using the java command followed by the name of the main class as: java Main
				    5. Enter valid values as prompted and press Enter key
